# Modern Flutter App

A new Flutter project with a modern UI setup.

## Getting Started

This project is a starting point for a Flutter application.

### Prerequisites

- Flutter SDK
- Dart SDK
- VS Code (recommended) with Flutter and Dart extensions

### Running the App

1.  Enable Developer Mode on Windows (Settings -> Update & Security -> For developers -> Developer Mode). This is required for symlinks used by Flutter plugins.
2.  Run `flutter pub get` to install dependencies.
3.  Run `flutter run` to start the app.

### Features

- Material 3 Design
- Google Fonts (Poppins)
- Modern Navigation Bar
- Responsive Layout
